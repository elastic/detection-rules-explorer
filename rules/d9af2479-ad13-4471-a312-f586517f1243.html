Curl or Wget Spawned via Node.js

Last updated a month ago on 2025-10-17
Created 2 months ago on 2025-09-18

About

This rule detects when Node.js, directly or via a shell, spawns the curl or wget command. This may indicate command and control behavior. Adversaries may use Node.js to download additional tools or payloads onto the system.

Query
process where host.os.type == "linux" and event.type == "start" and event.action in ("exec", "ProcessRollup2") and process.parent.name == "node" and (
  (
    process.name in ("bash", "dash", "sh", "tcsh", "csh", "zsh", "ksh", "fish") and
    process.args == "-c" and process.command_line like~ ("*curl*", "*wget*")
  ) or 
  (
    process.name in ("curl", "wget")
  )
)
